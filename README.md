# 🚀 Gverse - Social Media Web Application

Gverse is a modern **full-stack social media platform** where users can create profiles, share posts, interact with others, and explore a dynamic online community.
The project focuses on delivering a **smooth user experience with modern web technologies**.

---

## 🌟 Features

* 🔐 **User Authentication**

  * Sign Up / Login
  * Secure authentication system

* 👤 **User Profiles**

  * Create and manage personal profiles
  * Display user information

* 📝 **Post Creation**

  * Upload posts with images and captions
  * Share content with the community

* ❤️ **Social Interactions**

  * Like posts
  * Comment on posts

* 📱 **Responsive Design**

  * Works smoothly on desktop and mobile devices

* ⚡ **Dynamic UI**

  * Interactive and fast user interface built with React

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/DEVRAHUL-DEVELOPER/Gverse_Social_Media.git
```

### 2️⃣ Navigate to the project folder

```bash
cd Gverse_Social_Media
```

### 3️⃣ Install dependencies

Frontend:

```bash
cd client
npm install
```

Backend:

```bash
cd server
npm install
```

### 4️⃣ Run the application

Start backend server:

```bash
npm run dev
```

Start frontend:

```bash
npm run dev
```
